## 🔄 每天自動更新(不用你動手)

- 每個交易日會自動抓資料、更新網站。
- 它跑在 **GitHub 的伺服器**上,和你用哪台電腦、有沒有開機**都無關**。
- 免費排程常延到**傍晚(約 21 點)**才更新,屬正常現象。
- 週末、國定假日自動略過。

## ⚡ 想「提前」看今天的資料(手動)

**時機:下午 15:30 之後**(盤後資料已公布齊全)

1. 打開上面的「手動更新頁面」
   (或:GitHub → 上方 **Actions** → 左側點「每日盤後資料抓取與部署」)
2. 右邊按 **「Run workflow ▾」**
3. **手動指定日期** → 留空(代表今天)
4. **略過 30 分鐘重試** → 填 `true`
5. 按綠色 **「Run workflow」**
6. 約 20~30 秒變綠勾,再等 1~2 分鐘,網站就更新到當天

> 或者直接跟 Claude 說一句「**幫我更新今天的**」,由它幫你按。

## 🏠 換到另一台電腦(公司 ↔ 家裡)

- **看網站 / 手動更新**:任何電腦用瀏覽器登入 `github.com` 就能做,
  **不需要**本機那個資料夾、不需要裝 Git、也不需要 .bat 檔。
- **跟 Claude 接續**:登入同一個 Claude 帳號即可。
- **自動更新**完全不受影響,照常運作。
- 只有要「在新電腦上直接改程式」時,才需要先把 GitHub 專案複製(clone)下來——到時候請 Claude 帶你做。
